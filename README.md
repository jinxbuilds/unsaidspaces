# Unsaid Spaces

A minimal digital space for photography, technology, and public-awareness writing.

## Overview

Unsaid Spaces is a cinematic-style personal portfolio and visual journal focused on:

- Photography and curated visuals
- Medium articles and public-awareness content
- Tech projects and experiments
- Clean, distraction-free storytelling

The project is designed with a calm and minimal aesthetic while keeping performance and responsiveness in focus.

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Cloudinary
- Vercel

## Features

- Responsive modern UI
- Cloud-hosted image delivery
- Smooth gallery experience

## Local Setup

Clone the repository:

```bash
git clone https://github.com/jinxbuilds/unsaidspaces.git
```

Move into the project:

```bash
cd unsaidspaces
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

---

## Deployment

This project is deployed on Vercel.

Recommended deployment workflow:

- Push changes to GitHub
- Import repository into Vercel
- Configure environment variables if required
- Deploy
